# 🍔 iFood Online Ordering System

A web-based food ordering system developed using **PHP**, **MySQL**, **JavaScript**, and **CSS**.  
This application allows users to browse restaurants, order food online, and manage their orders, while admins can manage the entire system through a dashboard.

---

## 📌 Features

### 👤 User Features
- User registration and login
- Browse restaurants
- View food items (dishes)
- Add items to cart
- Place orders
- View order history
- Logout system

### 🛠️ Admin Features
- Admin dashboard
- Manage users
- Add/manage restaurants
- Manage food categories
- Add/manage menu items
- View and manage orders
- Delivery/order tracking management

## ⚙️ Installation Guide

### 🔧 Requirements
- XAMPP / WAMP / MAMP
- PHP 7+
- MySQL / MariaDB
- Web Browser

### 🚀 Setup Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/Group-Alpha-Project/Group-Project-iFood-Online-Ordering-System.git
   ```

2. Move the project folder to your server directory:
   - XAMPP → `htdocs/`
   - WAMP → `www/`

3. Open **phpMyAdmin**

4. Create a new database (e.g., `ifood_db`)

5. Import the SQL file from:
   ```
   DATABASE FILE/
   ```

6. Configure database connection:
   - Open files inside `connection/`
   - Update database name, username, and password

7. Start Apache & MySQL

8. Run the project in your browser:
   ```
   http://localhost/iFood Online Ordering System/
   ```
