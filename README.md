# 🍔 iFood Online Ordering System

A web-based food ordering system developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**.

The system allows customers to browse restaurants, order food online, and manage their orders. Administrators can manage restaurants, menu items, users, and orders through an admin dashboard.

---

## 📌 Features

### 👤 User Features

- User Registration
- User Login
- Browse Restaurants
- View Food Menu
- Add Items to Cart
- Place Orders
- View Order History
- Logout

### 🛠️ Admin Features

- Admin Dashboard
- Manage Users
- Manage Restaurants
- Manage Food Categories
- Manage Menu Items
- Manage Orders
- Order Status Management

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Group-Alpha-Project/Group-Project-iFood-Online-Ordering-System.git
```

### 2. Move the project

Copy the project folder into your local server.

**XAMPP**

```
htdocs/
```

**WAMP**

```
www/
```

### 3. Create the Database

Open **phpMyAdmin**

Create a database named:

```
ifood_db
```

Import the SQL file located in:

```
DATABASE FILE/
```

### 4. Configure Database Connection

Open the files inside:

```
connection/
```

Update the following:

- Database Name
- Username
- Password

### 5. Start the Server

Start:

- Apache
- MySQL

### 6. Run the Project

Open your browser and visit:

```
http://localhost/
```

If your project folder is named **iFood Online Ordering System**, open:

```
http://localhost/iFood%20Online%20Ordering%20System/
```
